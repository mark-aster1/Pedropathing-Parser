A quick an easy way to test pedro path visualizer trajectories. (.pp File to .java class Parser)



**Tutorial (step by step):**

1. Design your path on [https://visualizer.pedropathing.com/](url) and Download it using the download button in the top right

<img width="3022" height="1748" alt="87280" src="https://github.com/user-attachments/assets/10408961-523f-4eb8-ac89-317522585038" />

2. Open PP Parser
3. Click Browse
<img width="1668" height="88" alt="77458" src="https://github.com/user-attachments/assets/c9c239ae-c2a5-4725-ad4c-6823a5a28fa4" />

4. Select .pp File
<img width="1768" height="870" alt="59730" src="https://github.com/user-attachments/assets/de619026-2910-4f9f-9b0e-c8c5000efddf" />

5. Select a name for your Java class
<img width="702" height="58" alt="21420" src="https://github.com/user-attachments/assets/52990e89-16c2-44b7-9233-8e087820fcfa" />

6. Click Generate Java Code
7. Now you can either copy from the text box below or download the whole class from the button on the bottom of the screen.
 
<img width="1872" height="1026" alt="46929" src="https://github.com/user-attachments/assets/be53b7de-74b2-4137-9dd1-2a34b7cd2071" />
